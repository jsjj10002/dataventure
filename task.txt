# 기능 체크 및 프론엔드, 기능 고도화하기
[ 공통 기능 ]
1. 로그인, 회원가입
2. 구직자의 피드에는 회사들의 프로필 정보가, 채용 담당자의 피드에는 구직자의 프로필이 추천되어 뜸, 
3. 홈 회면에서 사용자, 회사 정보를 검색할 수 있다. 
4. 로딩은 스켈레톤 페이지 로딩 방식으로 구현
5. 기능들은 상단 네비게이션바로, 컨텐츠는 중앙에 배치, 여백이 너무 많지 않게 구현
6. 포인트 칼라를 정해 디자인
7. 백엔드의  기능들과 프론트앤드 각 컴포넌트들이 잘 연결되도록 신경쓰기
8. HTML, css를 예쁘게 전문가 처럼 잘 할 수 있는 ㅎ라이브러리나 패키지, 탬플릿 등을 많이 이용하여 효율적으로 UI/UX 향상시킬 수 있도록 함. 

[구직자 기능]
1. 회원 정보(프로필) 등록, 업데이트
   - 기본 정보 입력: 사진, 이름, 학력, 프로젝트 경험, 경력, 자기 소개, 보유 기술 등 링크드인 시스템과 동일하게 제작
   - 이력서 또는 포트폴리오 문서(docx, pdf 등 )업로드. 포트폴리오나 블로그 링크 추가하는 기능
   - 자신의 페이지 공유 기능-고유 url 생성(추후 기능) 
2. AI 인터뷰 
   - 화면 우측 상단 카메라로 얼굴 보이게 나옴
   - 실시간으로 음성에 반응하는 AI 3D 캐릭터 얼굴 화면에 띄워서 표정 반응형 실시간 대화(쉽게 구현할 수 있는 라이브러리 또는 API 탐색하여 넣기)
   - 인터뷰는 연습과 실전이 있음. 
   [인터뷰 연습모드]
   - 인터터뷰 전 등록한 프로필 내용을 토대로 진행할 질문 리스트 10개 중 5개 선택 또는 질문 따로 만들기
   - 채팅 모드와 음성모드 중 선택하여 진행 할 수 있음. 
   - 인터뷰 끝난 이후 나눴던 스크립트 생성, 스크립트 요약, 분석(항목별 점수) 생성
   [인터뷰 실전 모드]
   - 실전 모드는 음성으로만 가능
   - 실전 모드는 결과 분석 결과가 프로필에 업데이트 됨. -> 채용 담당자 입장에서 확인할 수 있음. 
3. 인터뷰 과정
   1. 시작 전 사용자의 프로필정보를 토대로 미리 질문을 생성
   2. 아이스브레이킹 -> 직무 공통 질문 -> 직무 특화 질문 순으로 진행
   3. 답변을 듣고 바로 다음 답변으로 넘어가는 것이 아닌, 이전 대화 내용을 통해 추가 직무역량 평가에 필요한 정보를 알아낼 수 있도록 꼬리질문을 이어감.
   4. 인터뷰 시간을 실전모드에서는 15분, 연습모드에서는 정한 시간 내에서(5분, 10분, 15분, 20분) 동안 진행, 화면 왼쪽 위 분:초 타이머 있음. 시간이 끝나면 입력중인 것 까지만 듣고 "인터뷰를 마치겠습니다."라고 나옴. 

4. 인터뷰 질문 생성 과정
   [공통 평가 항목]
   1. 초반: 아이스 브래이킹 용 간단한 대화몇 번 이어가기
   2. 자기소개 질문, 자기소개 이후 추가 꼬리 질문 1~2개 

   [직무 특별 항목]
   1. 직무는 아래와같이 경영관리, 영업, 전산 직무로 나뉜다. (1번부터 3번까지 각 직무별 우선 순위 평가항목)
{경영관리(전략기획, 회계/경리, 인사, 총무)
   {1. 정보 분석능력
   2. 문제 해결 능력
   3. 유연한 사고력}, 
영업(영업, 마케팅) 
   {1.  협상 및 설득 능력
   2. 유연한 사고력
   3.  정보 분석 능력}, 
전산(IT 개발, 개발 기획)
   {1.  IT 능력
   2.  문제 해결 능력
   3.  정보 분석능력 }}
   (평가요소: 정보분석능력, 문제 해결능력, 유연한 사고능력, 협상및 설득 능력,  IT 능력 5가지 항목으로 나눈다.)

   2. 사용자 프로필을 통해 평가 요소 별 난이도를 자동으로 설정한다.(추론 방식을 주는 별도의 체크리스트 형식의 기준을 두고 이에 맟춰 각 항목별 상, 중, 하 난이도 선정)
   3. 선정된 난이도에 맞게 질문을 생성함. 
   4. 이후 대화 하면서 답변의 내용에 따라 추가 질문을 할지, 말지, 어떤 내용을 할지정도 결정해서 추가질문을 1개정도 추가로 하거나 다음 질문으로 넘어간다. 

5. 질문 유형
   ===공통
   1. 일상 대화 2~3턴
   2, 자기소개 물어보기
   3. 프로필 내용에 대한 질문 1~2개
   ===직무 특별
   a. 정보 분석 능력: 
      - 주어진 데이터/통계자료를 통해서 관련된 생각을 말하기
      - 주어진 데이터의 상황, 통계자료 등에서 어떤 방법으로 인사이트 얻을 수 있는지 말하기
   b. 문제 해결능력
      - 실무 상황을 주고 어떻게 할 것인지 말하기
      - 가상의 서업 또는 프로젝트설명 후 돈 배분/인원 배분
   c, 유연한 사고 능력
      - 정답이 없는 상황이나 상충되는 목표 사이에서 최적의 균형점을 찾기 
      - 전혀 다른 분야의 지식이나 개념을 직무에 연결시키켜 말하기 
   d. 협상 및 설득 능력
      - 정해진 주장으로 설득하기
      - 가상의 상품 팔기
   e. IT 능력
      - 알고리즘이나 자료구조 등 내용에 원리에 대해 말하기
      - 간단한 프로젝트 설명을 주고 어떻게 구현 설계할 것인지 말하기
(./ExampleQuestion.csv에 질문 예시 있음. RAG프로세스로 비슷하게 질문 생성할 수 있므면 좋을 듯)

6. 평가 방법:
   - 대화가 전부 진행 항 후 평가를 진행함. 동기적으로 
   1. 공통 평가 항목인 "의사소통능력"은 전달력, 어휘 사용 적절성, 문제 이해력을 평가함
   2. 5가지 평가 요소별로 점수를 메김
   3. 요소별로 조합하여 어떤 직무가 어울리는지 랭킹을 메김
   4. ai 평가는 인터뷰 끝난 이후에 이루어 지고 끝날 때 까지 다른 일도 할 수 있게 함. 분석/평가 완료되면 상단 종모양 알림창을 통해 알림이 쌓임
   5. 인터뷰 평가, 분석결과는 따로 프로필이나 내 인터뷰 기록을 볼 수 있는 탭에서 볼 수 있음. 그 안에는 3가지 의사소통능력을 대표하는 전달력, 어휘사용 적절성, 문제 이해력이 가로로 눕혀진 바 차트 형식으로 표현하고 그 아래에는 5가지 직무 특별 항목에 대한 점수를 5각형 차트를 통해 볼 수있고, 그 아래에는 추천 직무 랭킹과 한 말을 근거로 설명이 텍스트로 적혀있음. 또 평가/분석표 맨 아래에는 스크립트 확인하기 버튼을 통해 스트립트를 확인할 수 있음 .
7. 지원서 넣기 :
   추천 직무를 모집하고있는 화사를 추천해 주고 지원서를 넣을 수 있음. 

[채용담당자]
1. 회사 소개 작성 - 구직자의 프로필 작성과 마찬가지, 회사 정보, 인재상, 회사소개, 원하는 직무 등을 작성할 수 있고, 고유의 페이지 링크를 통해 다른 사용자가 볼 수도 있다.
2. 구직 공고 올리기, 내리기 가능 - 직무별로 공고 올릴 수 있고, 내릴 수 있음. 
3. 어울리는 사용자 추천 
   - 공고를 낸 직무에 점수가 높은 사용자를 추천해줌. 
4. 지원자 대쉬보드
   - 자신의 회사에 지원한 지원자들을 리스트형식으로 대쉬보드로 볼 수 있고, 종합적으로 어떤 지원자가 있는지 3줄평도 함께 제시됨. 
   - 대쉬보드에 지원자의 평가항목별 평균 점수, 지원자 수 등 통계적인 정보가 시각적으로 보여짐.
   - 지원자 리스트에 프로필사진과 이름이 함께 뜨는 데 클릭하면 그 사용자의 프로필 페이지로 이동 가능
