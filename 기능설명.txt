# Flex AI Recruiter - 기능 설명서

## 📌 프로젝트 개요

**Flex AI Recruiter**는 AI 기반 인터뷰 시스템과 채용 매칭을 결합한 차세대 채용 플랫폼입니다.
구직자는 AI와의 실시간 음성 인터뷰를 통해 객관적인 역량 평가를 받고, 
채용 담당자는 데이터 기반으로 최적의 인재를 추천받을 수 있습니다.

**핵심 기술**: OpenAI GPT-5 모델 기반 AI 인터뷰 엔진

===============================================================================

1.1 회원 가입 및 로그인
------------------------
• 역할 선택: 구직자 / 채용담당자
• 이메일 기반 회원가입
• JWT 토큰 기반 보안 인증


1.2 맞춤형 피드 시스템
------------------------
• 구직자: 관심 직무의 채용 공고 및 회사 프로필 추천
• 채용담당자: 공고에 적합한 구직자 프로필 추천
• AI 기반 매칭 알고리즘으로 적합도 순 정렬


1.3 통합 검색
------------------------
• 구직자 프로필 검색 (이름, 기술, 경력)
• 회사 및 채용 공고 검색 (회사명, 직무, 위치)
• 실시간 필터링 지원


1.4 실시간 알림
------------------------
• 인터뷰 평가 완료 알림
• 지원서 접수 알림
• 추천 매칭 알림
• 상단 종 모양 아이콘으로 확인


1.5 현대적 UI/UX
------------------------
• Tailwind CSS 기반 반응형 디자인
• Skeleton 로딩으로 부드러운 사용자 경험
• 직관적인 상단 네비게이션 바
• 포인트 컬러(Primary Blue)를 활용한 일관된 디자인

                    [ 3. AI 인터뷰 프로세스 (상세) ]

3.1 인터뷰 시작 전 준비
------------------------
[자동 질문 생성 프로세스]

Step 1: 사용자 프로필 분석
• 학력, 경력, 보유 기술 스택 추출
• 희망 직무 식별 (경영관리 / 영업 / 전산)
• 프로젝트 경험 키워드 분석

Step 2: 난이도 자동 설정
• 경력 기간: 신입(하) / 3년 이하(중) / 3년 이상(상)
• 보유 기술: 개수 및 심화도 평가
• 프로젝트 경험: 복잡도 및 역할 평가

Step 3: 맞춤형 질문 풀 생성
• 평가 항목별 난이도에 맞는 질문 10개 생성
• GPT-5 모델이 사용자 맥락을 고려하여 커스터마이징


3.2 인터뷰 진행 단계
------------------------
[3단계 구조화된 인터뷰]

Phase 1: 아이스브레이킹 (2~3분)
• 간단한 일상 대화로 긴장 완화
• 예시: "오늘 날씨가 좋네요. 어떻게 오셨어요?"

Phase 2: 공통 질문 (5~7분)
• 자기소개 요청
• 지원 동기 및 경력 사항 확인
• 프로필 기반 꼬리 질문 1~2개

Phase 3: 직무 특화 질문 (7~10분)
• 선택된 직무의 우선 평가 항목 집중 질문
• 답변 내용에 따라 GPT-5가 실시간 꼬리 질문 생성
• 예시: "방금 말씀하신 프로젝트에서 가장 어려웠던 점은?"


[실시간 대화 흐름 제어]
• GPT-5가 대화 맥락을 유지하며 자연스러운 질문 생성
• 답변이 불충분하면 추가 질문 (최대 2개)
• 답변이 충분하면 다음 평가 항목으로 이동
• 시간 제한 내 최대 정보 수집 최적화


3.3 STT/TTS 시스템
------------------------
[음성 인식 (Speech-to-Text)]
• OpenAI Whisper API 활용
• 실시간 음성 → 텍스트 변환
• 한국어 특화 정확도 95% 이상

[음성 합성 (Text-to-Speech)]
• OpenAI TTS API (Nova 음성 모델)
• 자연스러운 한국어 발음
• 3D 아바타와 동기화된 립싱크

4.1 평가 항목 설계
------------------------
[공통 평가: 의사소통 능력 (Communication Skills)]

1) 전달력 (Delivery)
   측정 지표:
   • 문장 구조의 논리성 (주어-서술어 일치, 접속사 사용)
   • 답변 길이의 적절성 (너무 짧거나 길지 않음)
   • 핵심 키워드 포함 여부
   
   계산 방식:
   - GPT-5가 답변의 명확성을 0~100점으로 점수화
   - 구조화 점수(40%) + 길이 적절성(30%) + 키워드 점수(30%)

2) 어휘 사용 적절성 (Vocabulary)
   측정 지표:
   • 직무 관련 전문 용어 사용 빈도
   • 중복 표현 최소화
   • 격식 있는 어투 유지
   
   계산 방식:
   - 전문 용어 사용률(50%) + 어휘 다양성(30%) + 격식성(20%)

3) 문제 이해력 (Comprehension)
   측정 지표:
   • 질문 의도에 부합하는 답변
   • 요청하지 않은 정보 최소화
   • 후속 질문 필요 횟수
   
   계산 방식:
   - 의도 부합도(60%) + 정보 정확성(30%) - 추가 질문 패널티(10%)


[직무 특별 평가: 5대 역량]

직무별 우선순위:
• 경영관리: 정보 분석(1) > 문제 해결(2) > 유연한 사고(3)
• 영업: 협상·설득(1) > 유연한 사고(2) > 정보 분석(3)
• 전산: IT 능력(1) > 문제 해결(2) > 정보 분석(3)

1) 정보 분석 능력 (Information Analysis)
   평가 방법:
   • 데이터/통계 자료 해석 질문 제시
   • 제한된 정보에서 인사이트 도출 능력 측정
   
   예시 질문:
   "작년 대비 올해 매출이 20% 증가했지만, 고객 수는 5% 감소했습니다. 
    이 상황을 어떻게 해석하시겠습니까?"
   
   점수화:
   - 데이터 해석 정확성(40%)
   - 인사이트 창의성(30%)
   - 실행 가능한 제안(30%)

2) 문제 해결 능력 (Problem Solving)
   평가 방법:
   • 실무 시나리오 제시
   • 자원 배분, 우선순위 설정 능력 측정
   
   예시 질문:
   "프로젝트 마감 1주일 전에 핵심 개발자가 퇴사했습니다. 
    예산은 부족하고 일정은 촉박합니다. 어떻게 하시겠습니까?"
   
   점수화:
   - 문제 정의 명확성(30%)
   - 해결 방안 논리성(40%)
   - 리스크 관리(30%)

3) 유연한 사고 능력 (Flexible Thinking)
   평가 방법:
   • 정답 없는 상황에서 균형점 찾기
   • 다른 분야 지식을 직무에 연결
   
   예시 질문:
   "회사 성장을 위해서는 신규 시장 진출이 필요하지만, 
    기존 고객 만족도가 떨어질 위험이 있습니다. 어떻게 하시겠습니까?"
   
   점수화:
   - 다각도 사고(40%)
   - 창의적 연결(30%)
   - 실현 가능성(30%)

4) 협상·설득 능력 (Negotiation & Persuasion)
   평가 방법:
   • 특정 주장 설득하기
   • 가상 상품 판매 시나리오
   
   예시 질문:
   "예산 삭감을 반대하는 임원을 설득해야 합니다. 
    어떤 논리로 설득하시겠습니까?"
   
   점수화:
   - 논리 구조(40%)
   - 설득력(35%)
   - 근거 제시(25%)

5) IT 능력 (IT Skills)
   평가 방법:
   • 알고리즘/자료구조 원리 설명
   • 시스템 설계 사고 과정 평가
   
   예시 질문:
   "100만 명의 사용자 데이터를 빠르게 검색하려면 
    어떤 자료구조를 사용하시겠습니까? 그 이유는?"
   
   점수화:
   - 기술 이해도(40%)
   - 설계 논리성(35%)
   - 최적화 고려(25%)


4.2 평가 알고리즘
------------------------
[점수 계산 프로세스]

Step 1: 답변 텍스트 임베딩
• OpenAI Embedding API로 답변을 벡터화
• 의미론적 유사도 계산 가능

Step 2: 평가 기준 매칭
• 모범 답변과의 코사인 유사도 계산
• 키워드 매칭 점수 (TF-IDF)
• 답변 길이 및 구조 점수

Step 3: GPT-5 종합 평가
• 전체 대화 맥락을 고려한 종합 점수
• 항목별 강점/약점 텍스트 생성
• 개선 방향 제시

Step 4: 가중치 적용
• 직무별 우선순위에 따라 가중치 차등 적용
• 예: 경영관리 지원자는 정보 분석 점수 × 1.5

최종 점수 = 
  (의사소통 평균 × 0.3) + 
  (직무 역량1 × 0.25 × 우선순위1 가중치) +
  (직무 역량2 × 0.20 × 우선순위2 가중치) +
  (직무 역량3 × 0.15 × 우선순위3 가중치) +
  (기타 역량 × 0.10)


4.3 추천 직무 랭킹
------------------------
[매칭 알고리즘]

Step 1: 역량 프로필 생성
• 5개 역량 점수 → 벡터 [A, B, C, D, E]
• 정규화 (0~100 스케일)

Step 2: 직무별 적합도 계산
각 직무의 요구 역량 가중치와 내적(Dot Product) 계산

경영관리 적합도 = 
  (정보 분석 × 0.35) + (문제 해결 × 0.30) + 
  (유연한 사고 × 0.25) + (협상·설득 × 0.05) + (IT × 0.05)

영업 적합도 = 
  (협상·설득 × 0.40) + (유연한 사고 × 0.30) + 
  (정보 분석 × 0.20) + (문제 해결 × 0.05) + (IT × 0.05)

전산 적합도 = 
  (IT × 0.45) + (문제 해결 × 0.30) + 
  (정보 분석 × 0.20) + (유연한 사고 × 0.03) + (협상·설득 × 0.02)

Step 3: 랭킹 생성 및 이유 제시
• 적합도 점수 순으로 정렬
• GPT-5가 각 직무 추천 이유를 자연어로 생성
  예: "정보 분석 능력이 상위 10%로, 데이터 기반 의사결정이 
       중요한 전략기획 직무에 적합합니다."


4.4 통계적 벤치마킹
------------------------
[상대 평가 시스템]

• 전체 사용자 대비 백분위 계산
  예: "정보 분석 능력이 상위 15%입니다."

• 동일 직무 지원자 대비 비교
  예: "전산 직무 지원자 평균보다 12점 높습니다."

• 시계열 분석 (연습 모드 활용 시)
  - 이전 인터뷰 대비 점수 변화 추적
  - 개선된 항목 / 개선 필요 항목 시각화

6.3 AI 기반 인재 추천
------------------------
[매칭 알고리즘]

Step 1: 공고 분석
• 직무 카테고리 추출
• 필수/우대 기술 키워드 추출
• 요구 역량 프로필 생성

Step 2: 후보자 풀 검색
• 해당 직무 실전 인터뷰 완료자 필터링
• 기술 스택 일치도 계산
• 역량 점수 가중치 적용

Step 3: 매칭 점수 계산
매칭 점수 = 
  (직무 적합도 × 0.40) + 
  (기술 스택 일치도 × 0.30) + 
  (종합 평가 점수 × 0.20) + 
  (경력 적합성 × 0.10)

Step 4: 추천 이유 생성
• GPT-5가 각 후보자의 강점을 요약
• 공고 요구사항과의 매칭 포인트 강조

예시:
"김철수 님은 React, TypeScript 실무 경력 3년으로 
 귀사의 프론트엔드 개발자 요구사항에 95% 부합합니다. 
 특히 문제 해결 능력(92점)과 IT 능력(88점)이 우수합니다."
